URL-Shortener это сервис, который позволяет создавать короткие ссылки для длинных URL-адресов.

Для создания или удаления короткой ссылки используются HTTP запросы POST и DELETE с авторизацией.

Сервис работает на базе PostgreSQL.

Также настроен автоматический деплой на сервер с помощью GitHub Actions и Docker Compose.
